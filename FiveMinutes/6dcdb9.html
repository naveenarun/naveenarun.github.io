In most programming languages, the main medium of communication is text. Javascript is no different. Often, programmers find themselves working with text more than they work with pictures.

Javascript likes to be silent. You need to "politely" ask it to talk using "console.log". Whatever it says will be logged in a console (white box).

Task: Make Javascript talk to you using "console.log(statement)". Hit run, and feel free to experiment with different phrases. I'll explain the rest on the next page.
