So what just happened? What does Javascript even do????
On the previous page, you used Javascript to "log" (write) the phrase "hello world!" in the console. The console is a place where the computer can say things for you to see.

Javascript is used to program the web. For example, this website uses Javascript to add pretty colors your code in the text box and run it. Other websites use Javascript to make animations and interactive content. More recently, it has also been used for a bunch of things outside the web like apps and robots.

Web programmers use their browser's built-in console to see what their code does. Your browser has a console, but it is hidden along with other tools that browsers provide to web programmers. On this page, I have redirected all console output to the white box so that you don't have to bother with developer tools.

What happened on the previous page? The phrase "console.log()" uses parentheses because it's a function, much like how f(x)=x+2 is a function. It takes an input (x), does something to it (adds two to x), and gives an output (the new value of x). In this case, Javascript takes an input ("hello world!"), does something to it (writes it in the console), and gives an output (nothing).

Why are there semicolons? Javascript doesn't really require any white space (spaces, tabs, or carriage returns), so the only way it can know where new lines start is with semicolons. If you're using a modern browser, the semicolons aren't necessary as long as you press enter between lines. The editor will tell you if you have any problems with semicolons by putting a small "i" next to line numbers.
